# TEMS DocType Reference

This document lists custom DocTypes provided by the TEMS app with their fields to aid API/CRUD development.

Generated by `scripts/generate_doctype_docs.py`.

## Using with Frappe REST API

Base URL: `/api/resource/{DocType}`

Authentication: Use API Key/Secret headers or session cookie.

Common operations:

- List: `GET /api/resource/{DocType}?fields=["name",...]&filters=[["field","=","value"]]&limit_start=0&limit_page_length=20&order_by=modified desc`

- Get one: `GET /api/resource/{DocType}/{name}`

- Create: `POST /api/resource/{DocType}` with JSON body `{"fieldname": value, ...}`

- Update: `PUT /api/resource/{DocType}/{name}` with JSON body for fields to update

- Delete: `DELETE /api/resource/{DocType}/{name}`

Notes: For Link fields, pass the linked document's name. For Table fields (child tables), pass a list of child row dicts under the child fieldname.
