# TEMS PWA — LANDING PAGE PROMPT

## ROLE
You are designing the **Landing Page** for the TEMS Progressive Web Application suite.  
The page serves as the public gateway and internal PWA entry portal for all roles (Driver, Operations, Safety, Finance, Admin).

---

## TASKS

1. **Hero Section**
   - Title: "TEMS — Transport Excellence Management System"
   - Subtext: “Fleet. Operations. Safety. Finance. Insights — all in one platform.”
   - Buttons:
     - “Driver Portal”
     - “Operations Portal”
     - “Safety Portal”
     - “Admin Login”

2. **Feature Highlights**
   - Cards for:
     - Fleet Management
     - Cargo Logistics
     - Passenger Transport
     - Real-Time Operations
     - Safety & Compliance
     - Financial Insights
   - Animated icons and short descriptions.

3. **Live Metrics Section**
   - Show total vehicles, trips, consignments, and passenger count (fetched from API).

4. **Install as PWA Prompt**
   - “Install TEMS on your device” button (Add to Home Screen).
   - Instructions for mobile and desktop.

5. **Footer**
   - Company info (Infometics Solution / TEMS)
   - Links: Privacy, Support, Terms of Service.

---

## CONSTRAINTS
- Clean, responsive layout.
- Built with Next.js, React, or Flutter Web.
- Branding consistent with TEMS UI (navy, teal, and white color scheme).
- SEO-optimized (meta tags, manifest.json).
- Should auto-detect user role post-login and redirect to appropriate PWA.

---

## OUTPUTS
- Hero section wireframe and style tokens.
- Feature grid component.
- API call example for real-time metrics.
- Manifest.json configuration and service worker setup.
- Entry routes for /driver, /operations, /safety.
