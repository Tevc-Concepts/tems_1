## Starting a Journey

### Pre-Trip Checklist

**Before Starting ANY Journey:**

**1. Vehicle Inspection (MANDATORY)**
- Open "Inspections" â†’ "New Inspection"
- Select "Pre-Trip Inspection"
- Complete all checklist items (see Vehicle Inspections section)
- Submit inspection
- âœ… Pass â†’ Proceed with journey
- âŒ Fail â†’ Report issues, DO NOT DRIVE

**2. Verify Journey Details**
- Open journey from schedule
- Confirm:
  - Route correct
  - Departure time correct
  - Vehicle assigned is available
  - Special instructions read

**3. Check Documents**
- Driver's license (valid)

  - Insurance

### Starting the Journey

**Steps:**
1. Open journey from schedule
2. Verify you completed pre-trip inspection
3. Click **"Start Journey"** button
4. System prompts:
   - Confirm start time (auto-filled with current time)
   - Confirm odometer reading (IMPORTANT: Enter accurately)
   - Confirm location
5. Optional: Add photo of odometer
6. Click "Confirm Start"

**Result:**
- Journey status: **IN PROGRESS**
- GPS tracking activated (if available)
- Operations team notified
- Journey timer started

**IMPORTANT:** Always start journey in system when physically starting. Accurate timing is critical for reporting.

- âœ… Obey all traffic laws
- âœ… Drive defensively
- âœ… Stay within speed limits
- âœ… Use seatbelt
- âœ… Keep safe following distance
- âœ… Be courteous to other drivers and passengers

- âŒ Drink alcohol before or during duty
- âŒ Use drugs that impair driving

### Managing Passengers (If Applicable)

**Boarding:**
- Greet passengers politely
- Assist elderly, disabled, or passengers with luggage
- Ensure all seated before moving
- Make safety announcements if required

**During Transit:**
- Drive smoothly (avoid sudden brakes/turns)
- Announce major stops
- Be helpful if passengers have questions
- Maintain professional demeanor

**Alighting:**
- Come to complete stop
- Give passengers time to disembark safely
- Assist as needed

**Scheduled Stops:**
- Follow route schedule for stops
- Don't skip authorized stops
- Minimal delay at each stop
- Record stop in app if required

**Unscheduled Stops:**
- Only for emergencies or necessity
- If need to stop (breakdown, medical emergency, etc.):
  1. Find safe location
  2. Turn on hazard lights
  3. Update journey status in app
  4. Contact supervisor/operations
  5. Document reason

### Recording Journey Updates

**While Safe to Do So:**
- Update journey status if delayed
- Report traffic conditions
- Note any incidents
- Record stops made

**DO NOT** use app while vehicle is moving. Pull over safely or wait until stopped.

## Completing a Journey

### End of Journey Process

**When You Reach Final Destination:**

**1. Post-Trip Inspection**
- Inspect vehicle for any new damage
- Check for passenger items left behind
- Note any issues that developed during trip

- Open active journey
- Click **"Complete Journey"** button

  - **End Time:** (auto-filled, confirm accurate)
  - **Final Odometer Reading:** (IMPORTANT: Enter accurately)
  - **End Location:** (confirm or adjust)

**3. Journey Summary**
- Review journey summary:
  - Duration
  - Distance
  - Average speed
  - Route taken (map)
- Correct if any errors

**4. Report Any Issues**
- If any incidents occurred:

  - Click "Report Vehicle Issue"

**5. Passenger Count (If Applicable)**
- Enter number of passengers carried

**6. Expenses (If Any)**
- Add any expenses incurred:
  - Tolls
  - Parking
  - Emergency repairs (if authorized)
  - Other authorized expenses
- Upload receipts

**7. Final Notes**
- Add any additional notes
- Observations
- Recommendations

**8. Submit Completion**
- Review all information
- Click "Submit"
- Journey status: **COMPLETED**
- Operations team notified

  - Prepare for next trip

**End of Day:**
- All journeys completed

- Keys returned (if required)

## Reporting Incidents

### Types of Incidents

**1. Traffic Accidents**
- Collision with another vehicle
- Hit fixed object
- Hit pedestrian/animal
- Single-vehicle accident

**2. Vehicle Breakdowns**
- Engine failure
- Flat tire
- Electrical problem
- Any issue preventing operation

**3. Safety Incidents**
- Passenger injury
- Passenger complaint
- Near-miss incident
- Unsafe condition

**4. Security Incidents**
- Theft/vandalism
- Threatening behavior
- Suspicious activity
- Missing items

### Immediate Actions in Incident

**Priority: Safety First**

**If Accident:**
1. **Stop immediately** - Do not leave scene
2. **Check for injuries** - Attend to injured
3. **Call emergency services** if needed (ambulance, police)
4. **Secure the scene:**
   - Turn on hazard lights
   - Set up warning triangle
   - Move vehicle if blocking traffic (if safe and legal)
5. **Contact supervisor/operations immediately**
6. **Do NOT admit fault** - State facts only
7. **Exchange information** with other parties:
   - Names, addresses, phone numbers
   - Vehicle registrations
   - Insurance details
   - Witness information
8. **Document the scene:**
   - Take photos (vehicles, damage, scene, road conditions)
   - Note time, location, weather, lighting
   - Draw simple diagram if possible
9. **Wait for authorities** if required
10. **Report in TEMS system** as soon as safe to do so

**If Breakdown:**
1. **Get vehicle to safe location** if possible
2. **Turn on hazard lights**
3. **Set up warning triangle**
4. **Contact operations for assistance**
5. **Stay with vehicle** (unless unsafe)
6. **Keep passengers informed and safe**
7. **Report in system**

**Steps:**

**3. Parties Involved**

  - Name

  - Contact info

**4. Damage/Injuries**
- Vehicle damage:
  - Describe
  - Estimate severity

  - Who injured

  - Medical attention sought?
- Property damage (if any)

**5. Documentation**
- Upload photos:
  - Vehicle damage (multiple angles)
  - Other vehicles involved
  - Scene (wide view)
  - Road conditions

  - Police report (if available)

## Troubleshooting

**Q: Cannot start journey**  
A: Verify pre-trip inspection completed and passed. Check journey is assigned to you. Ensure app has internet connection.

**Q: GPS not working**  
A: Enable location services for app in phone settings. If issue persists, journey can still be completed manually.

**Q: Journey not appearing in schedule**  
A: Refresh app (pull down on screen). Check date selected. Contact supervisor if still missing.

**Q: Cannot submit inspection**  
A: Ensure all checklist items marked (Pass or Fail). Check internet connection. Try again after a moment.

**Q: Forgot to start journey**  
A: Start journey as soon as realized. Enter actual start time in notes. Notify supervisor.

**Q: App not working**  
A: Check internet connection. Try closing and reopening app. Clear app cache if available. Contact IT support if persists.

## FAQ

**Q: What if I arrive late to start shift?**  
A: Notify supervisor immediately. Document reason. Start journey when ready but note delay.

**Q: Can I swap shifts with another driver?**  
A: Must get supervisor approval. Cannot simply swap without authorization in system.

**Q: What if vehicle assigned has issues I find during inspection?**  
A: Complete inspection marking issues. Report to supervisor immediately. Do NOT drive if critical safety issue. Await alternate vehicle.

**Q: What if passenger behaves aggressively?**  
A: Prioritize safety. Stop vehicle in safe location if necessary. Contact supervisor/security immediately. Report incident in system. Do not engage in confrontation.

**Q: Can I deviate from assigned route?**  
A: Only with supervisor permission due to road closure, emergency, or operational necessity. Document reason in journey notes.

**Q: What if I feel too tired to drive safely?**  
A: Do NOT drive. Notify supervisor immediately. Fatigue is major safety risk. Company policy supports safe operations over schedule.

**Q: What if my license is about to expire?**  
A: Renew immediately. Cannot drive with expired license. Upload renewed license to system as soon as received.

**Q: How do I contact someone in emergency?**  
A: Emergency contacts available in app under "Emergency Contacts". Includes operations center, supervisor, security, management.
